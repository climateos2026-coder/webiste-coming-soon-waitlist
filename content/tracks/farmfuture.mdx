## Why FarmFuture Matters

By 2080, staple crop yields across tropical regions could fall by one-third. Most adaptation tools are built for large industrial farms in the Global North. Smallholder farmers need affordable, offline-capable tools that work on basic smartphones.

## The Challenge

Build an open-source tool that helps a smallholder farmer decide when to plant their next crop cycle using seven-day weather forecasts, satellite soil moisture data, and historical yield patterns.

## What Success Looks Like

- A farmer opens your tool on a $50 Android phone
- Gets a simple yes/no recommendation in under 5 seconds
- Tool works offline with weekly data sync
- Open-source and documented in five languages

## Datasets Provided

| Dataset | Source | Format | Notes |
|---------|--------|--------|-------|
| Soil moisture | NASA SMAP | GeoTIFF / CSV | Global, 2015-2025 |
| Weather forecast | open-meteo | JSON API | Free, no key required |
| Crop calendars | FAO | CSV | 195 countries |
| Yield data | World Bank | CSV | 2000-2024 |
| Market prices | FAO | API | 180+ countries |
| Satellite imagery | Sentinel-2 | GeoTIFF | 10m resolution |
