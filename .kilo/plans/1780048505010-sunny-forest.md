# Comprehensive Code Audit and Refactor Plan for ClimateOS 2026

## Executive Summary

Audit of ClimateOS 2026 website codebase reveals issues in security, accessibility, code quality, and deployment configuration. The site uses Next.js 16 with Tailwind CSS 4, Supabase, and custom components.

---

## Critical Issues (Must Fix Before Deployment)

### 1. Security: Deprecated X-XSS-Protection Header
**File:** `next.config.ts:65-67`
- The `X-XSS-Protection` header is deprecated and ignored by modern browsers
- **Fix:** Remove this header entirely

### 2. Security: Missing Permissions-Policy Header
**File:** `next.config.ts`
- Missing `Permissions-Policy` header to restrict browser features
- **Fix:** Add Permissions-Policy to disable unnecessary browser APIs

### 3. Security: Missing Cache-Control for Static Assets
**File:** `next.config.ts`
- No explicit caching headers for static assets
- **Fix:** Add appropriate cache headers for static assets

---

## High Priority Issues

### 4. Component: ParticipantCounter Memory Leak
**File:** `components/sections/participant-counter.tsx:25-37`
- Creates Supabase client inside useEffect, potential for multiple subscriptions
- **Fix:** Add abort controller and prevent race conditions

### 5. Component: WaitlistForm XSS Vulnerability
**File:** `components/forms/waitlist-form.tsx`
- Uses inline `select` element instead of Select component
- Inline styles duplicated across form fields
- **Fix:** Refactor to use UI components consistently

### 6. Bug: CSRF Header Naming
**File:** `components/forms/waitlist-form.tsx:60` and `app/api/waitlist/route.ts:100`
- Form sends `X-CSRF-Token`, API reads `x-csrf-token` (case inconsistency in headers)
- **Fix:** Standardize to lowercase header name

### 7. Accessibility: Navbar Missing ARIA
**File:** `components/layout/navbar.tsx`
- Mobile menu button missing `aria-expanded` and `aria-controls`
- Mobile menu panel missing `id` for aria-controls reference
- **Fix:** Add proper ARIA attributes

### 8. Accessibility: FAQ Accordion Roles
**File:** `app/faq/page.tsx:103-115`
- Custom accordion buttons missing `aria-expanded`, `aria-controls`, `role="button"`
- **Fix:** Add proper accordion ARIA pattern

### 9. Accessibility: Missing Skip Link
**File:** `app/layout.tsx`
- No skip-to-content link for keyboard navigation
- **Fix:** Add skip link in layout

### 10. Performance: ClimateGlobe High Particle Count
**File:** `components/sections/climate-globe.tsx:88`
- 2000 particles may cause performance issues on mobile
- **Fix:** Reduce to 1200 or implement device-based scaling

## Medium Priority Issues

### 11. Dead Code: HeroSection Component
**File:** `components/sections/hero.tsx`

### 12. Broken Links: ClimateTicker
**File:** `components/sections/climate-ticker.tsx:26`
- Links to `/tracks/${stat.trackSlug}` but no dynamic track pages exist
- **Fix:** Remove links or make them non-clickable spans

### 13. Missing Fonts
**File:** `public/fonts/` and `app/globals.css`

### 14. Vercel Configuration Incomplete
**File:** `vercel.json`
